# PPT Editor

A desktop PowerPoint editor built using React, Electron, and Tailwind CSS.

## Features
- Create and edit slides
- Add text and elements
- Simple UI for presentations
- Desktop application using Electron

## Technologies Used
- React
- Electron
- Tailwind CSS
- Vite

## Installation

Clone the repository:

git clone https://github.com/Suma-Hiremath/ppt-editor.git

Install dependencies:

npm install

Run the project:

npm run dev
